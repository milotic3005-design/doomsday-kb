Core Architecture

Built for the end of the internet.

Six hardened subsystems. Zero cloud dependencies. Every answer grounded in your knowledge base — not hallucinated from training data.

RAG-Grounded Intelligence

Every response is retrieved from your curated knowledge base — not generated from stale training data. Vector similarity search finds the exact passage, injects it as context, and the LLM cites its source. Zero hallucination by design.

Offline LLM Engine

Auto-selects the optimal model for your hardware. Llama 3.2 3B on 8GB machines, Qwen 3.5 9B with chain-of-thought reasoning on 16GB+ — all running locally via Ollama.

Modular Knowledge Packs

Six domain packs — medical, agriculture, engineering, comms, security, pharmacy — each installable and removable without rebuilding the application.

Semantic Vector Search

768-dimension embeddings with LanceDB similarity matching across your entire knowledge base.

Air-Gapped Security

Zero network calls after setup. No telemetry, no cloud sync, no subscriptions. Fully sovereign.

Chain-of-Thought Reasoning

Qwen 3.5 native thinking mode — visible reasoning chain before every answer on critical queries.
